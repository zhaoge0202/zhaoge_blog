# 算法

## 为什么重要

算法考的不是背题，而是复杂度意识和把问题抽象成模型的能力，这恰恰是工程取舍的底层素养。

## 知识主线

复杂度分析 -> 基础算法思想 -> 高频题型 -> 实战套路

## 后续计划

- 时间与空间复杂度分析。
- 排序、二分、双指针、滑动窗口。
- 递归、回溯、贪心、分治。
- 动态规划常见模型。
- 链表、树、图的高频面试题型。

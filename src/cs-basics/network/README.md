# 计算机网络

## 为什么重要

接口、RPC、网关、长连接这些工程问题，最终都要回到 TCP/IP 和 HTTP 的基本原理。

## 知识主线

网络分层模型 -> TCP/IP -> HTTP/HTTPS -> 应用层协议

## 后续计划

- TCP 三次握手、四次挥手、可靠传输、流量控制与拥塞控制。
- TCP 与 UDP 的区别与选型。
- HTTP 报文结构、状态码、长连接、HTTP/1.1 与 HTTP/2。
- HTTPS 加密流程、对称/非对称加密、证书校验。
- DNS 解析、CDN、跨域。

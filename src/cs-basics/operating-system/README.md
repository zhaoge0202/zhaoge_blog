# 操作系统

## 为什么重要

线程模型、锁、IO、JVM 内存这些 Java 高频题，往下追问几层都会落到操作系统原理上。

## 知识主线

进程与线程 -> 内存管理 -> IO 模型 -> 调度与同步

## 后续计划

- 进程、线程、协程的区别与切换成本。
- 虚拟内存、分页分段、缺页中断。
- 用户态与内核态、系统调用、零拷贝。
- 阻塞/非阻塞、同步/异步、IO 多路复用（select / poll / epoll）。
- 死锁的产生条件与排查。

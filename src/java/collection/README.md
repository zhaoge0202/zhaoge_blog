# 集合

## 为什么重要

HashMap、ArrayList 的源码是面试最高频的考点之一，背后牵出数据结构、扩容、并发安全一连串追问。

## 知识主线

List -> Map -> Set -> 并发集合

## 后续计划

- ArrayList 与 LinkedList、扩容机制。
- HashMap 原理、扩容、红黑树化、线程安全问题。
- ConcurrentHashMap 的演进与实现。
- HashSet、TreeMap、LinkedHashMap。
- fail-fast 与 fail-safe。

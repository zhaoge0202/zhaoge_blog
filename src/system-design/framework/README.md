# 框架

## 为什么重要

Spring 全家桶是 Java 后端的事实标准，IoC、AOP、事务、自动装配几乎是必问，还能直接连到项目经验。

## 知识主线

Spring 核心 -> Spring MVC -> Spring Boot -> Spring Cloud / MyBatis

## 后续计划

- Spring：IoC、AOP、Bean 生命周期、循环依赖。
- 事务传播机制与失效场景。
- Spring MVC 请求处理流程。
- Spring Boot 自动装配原理。
- MyBatis 工作原理与缓存。

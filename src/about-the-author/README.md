# 关于作者

你好，我是 `zhaoge`。这个站点不是为了把外部资料机械搬运进来，而是为了把 Java 后端面试、开发实践和个人复盘整理成一套长期可维护的知识库。

## 我希望这个站点解决什么问题

- 知识点不要只停留在“标准答案”，而要能落回项目里的取舍和排障。
- 博客和知识库要分开，避免实战记录把主线打散。
- 内容要持续纠偏，而不是一次性堆满。

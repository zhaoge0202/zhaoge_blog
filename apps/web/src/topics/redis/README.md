Redis 高频出现在性能优化、缓存一致性和分布式协调场景。

## 知识主线

数据结构 -> 持久化 -> 过期淘汰 -> 缓存问题 -> 高可用 -> 分布式锁

## 面试焦点

能讲清缓存问题治理和真实项目取舍。

## 前置知识

缓存基础、网络 IO 基础

## 目标人群
